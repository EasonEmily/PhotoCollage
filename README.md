# Photo Collage Web

> AI Photo Collage - 用 Next.js + Tailwind CSS 构建的合影生成平台

## 技术栈

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel Ready

## 功能

- 📤 照片上传（支持拖拽、点击上传）
- 🎭 5 种合照模板可选
- ⚙️ AI 合成进度展示
- 📥 预览与下载
- 📖 使用教程页面
- 🔒 隐私政策页面

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 启动
npm start
```

## 页面结构

- `/` - 首页
- `/upload` - 上传页面
- `/generate` - 生成中页面
- `/result` - 结果页面
- `/tutorial` - 使用教程
- `/privacy` - 隐私政策
