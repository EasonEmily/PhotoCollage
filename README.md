# AI Photo Collage

> 把多张单人照合成逼真合照 — AI 让合照不再受距离限制

## 产品概述

用户上传多张单人照片，AI 自动将人物融合到一张合影图中，生成自然逼真的群体合照。

**核心价值**：解决"想合照但人凑不齐"的场景痛点，让用户轻松获得"完美合照"。

## 功能范围（MVP）

### 核心功能
- [ ] 照片上传（支持拖拽/JPG/PNG/WebP，单次最多6张）
- [ ] AI 合成引擎（人脸检测 + 图像融合）
- [ ] 模板选择（室内合影/户外背景/团队站立照等）
- [ ] 生成与下载
- [ ] 用户引导

### 辅助功能（后续迭代）
- [ ] 用户登录 / 历史记录
- [ ] 批量生成
- [ ] 自定义背景
- [ ] 照片美颜/滤镜

## 技术方案

### 方案 B：调用第三方 AI API（适合 MVP 快速验证）

| 层级 | 技术选型 |
|------|---------|
| 前端 | Next.js / React |
| 后端 | Node.js / 云函数 |
| AI 合成 | 旷视 Face++ 人脸融合 / 腾讯云人脸融合 API |
| 存储 | 阿里云 OSS |
