# AI Photo Collage — MVP 需求文档

> 版本：v1.0 | 日期：2026-03-25 | 状态：初稿

---

## 1. 产品概述

**产品名称**：（待定，建议名：GroupSnap / CollageAI / 合照神器）

**一句话描述**：用户上传多张单人照片，AI 自动将人物融合到一张合影图中，生成自然逼真的群体合照。

**核心价值**：解决"想合照但人凑不齐"的场景痛点，让用户轻松获得"完美合照"。

**目标用户**：
- C端：想要生成家庭照、朋友照、团队照但无法线下聚齐的用户
- 运营/市场：需要快速生成多样化合照素材

---

## 2. 用户故事（User Stories）

| # | 角色 | 场景 | 需求 |
|---|------|------|------|
| U1 | 普通用户 | 春节想发家庭合照，但家人散落各地 | 上传各自单人照，获取一张完整家庭合照 |
| U2 | 普通用户 | 朋友不在同一个城市 | 把各自的照片合成一张"闺蜜合照" |
| U3 | 企业市场 | 需要大量不同团队的"合影"素材 | 批量上传，批量生成 |
| U4 | 用户 | 第一次使用 | 不知道如何操作，希望有引导教程 |

---

## 3. 功能范围（MVP）

### 3.1 核心功能

- [ ] **照片上传**
  - 支持拖拽上传 / 点击上传
  - 支持 JPG / PNG / WebP
  - 单次最多上传 6 张单人照
  - 每张照片不超过 10MB
  - 前端预览已上传照片

- [ ] **AI 合成引擎（后端）**
  - 检测每张照片中的人脸
  - 提取人脸特征和姿态
  - 将多个人物自然融合到一张目标合照模板中
  - 光照/肤色/角度智能适配

- [ ] **模板选择**
  - 提供 3-5 种预设合照场景模板（如：室内合影、户外背景、团队站立照）
  - 默认模板自动适配

- [ ] **生成与下载**
  - 生成进度提示
  - 支持预览合成结果
  - 一键下载（JPG/PNG）
  - 分享链接（可选）

- [ ] **用户引导**
  - 首页简短操作指引（3步完成）
  - 示例对比图（ Before / After ）

### 3.2 辅助功能（MVP 后迭代）

- [ ] 用户登录 / 历史记录（手机号/微信快捷登录）
- [ ] 批量生成
- [ ] 自定义背景
- [ ] 照片美颜/滤镜
- [ ] 去除水印（付费）

## 4. 非功能需求

### 4.1 性能

| 指标 | 要求 |
|------|------|
| 页面加载时间 | ≤ 3 秒 |
| AI 合成耗时 | ≤ 30 秒（单次） |
| 支持并发 | ≥ 50 人同时使用 |

### 4.2 稳定性

- 合成失败时提供友好错误提示
- 支持失败后重试

### 4.3 安全与隐私

- 用户上传照片仅用于当次合成，24h 内自动删除
- 隐私政策公示（必选）
- 敏感人脸数据不透出

### 4.4 兼容

- 浏览器：Chrome / Safari / Edge 最新版
- 移动端：iOS 14+ / Android 10+
- 响应式布局

## 5. 技术方案建议

### 方案 A：自研 AI 模型（推荐给有技术团队的企业）

| 层级 | 技术选型 |
|------|---------|
| 前端 | Next.js / React + TailwindCSS |
| 后端 | Python FastAPI / Node.js |
| AI 引擎 | InsightFace + Stable Diffusion / ComfyUI |
| 人脸检测 | RetinaFace / MediaPipe |
| 图像融合 | Diffuse to Blend / IP-Adapter |
| 存储 | 阿里云 OSS / 腾讯云 COS |
| GPU 服务器 | 标配 NVIDIA T4 起 |

**优点**：模型可控，效果可调
**缺点**：研发成本高周期长

---

### 方案 B：调用第三方 AI API（适合 MVP 快速验证）

| 层级 | 技术选型 |
|------|---------|
| 前端 | Next.js / React |
| 后端 | Node.js / 云函数 |
| AI 合成 | 旷视 Face++ 人脸融合 / 腾讯云人脸融合 API |
| 存储 | 阿里云 OSS |

**优点**：快速上线，技术门槛低
**缺点**：受制于第三方能力，定制化受限

---

### 方案 C：开源模型 + 云部署（适合独立开发者）

| 层级 | 技术选型 |
|------|---------|
| 前端 | React / Vue |
| 后端 | Python FastAPI |
| AI 模型 | FaceFusion（开源）+ Replicate 云 GPU |
| 存储 | Cloudflare R2 |

**优点**：成本可控，效果不错
**缺点**：需要一定技术能力
